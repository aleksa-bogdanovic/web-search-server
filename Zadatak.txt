Kreirati Web server koji klijentima preko browser-a omogućava pretraživanje reči u tekstualnim fajlovima. Svi zahtevi serveru se šalju preko browser-a korišćenjem GET metode. U zahtevu se kao parametri navode reči za koje se vrši pretraga. Kao odgovor klijentu, server šalje rezultat pretrage u html obliku, gde su prikazani nazivi fajlova koji sadrže tražene reči i broj pojavljivanja reči u fajlovima. Pretragu vršiti samo za fajlove koji se nalaze u root direktorijumu servera. Primer poziva serveru: http://localhost:5050/sistemsko&programiranje&elfak&projekat
strategija upravljanja kes memorijom je ogranicenje velicine.
